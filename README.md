# 💈 Muru Barbearia - Website

Este é um projeto de landing page moderna e responsiva desenvolvida para a **Muru Barbearia**. O site foi construído focado em uma experiência de usuário elegante, utilizando uma paleta de cores escura (Dark Mode) para transmitir sofisticação e profissionalismo

---

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as melhores práticas de desenvolvimento front-end:

* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização personalizada e efeitos de rotação.
* **Bootstrap 5**: Sistema de grid, cards, carrossel e componentes responsivos.
* **Google Fonts**: Utilização das fontes *Playfair Display* e *Montserrat*.

---

## ✨ Funcionalidades

* **Design Responsivo**: O layout se adapta perfeitamente a celulares, tablets e computadores.
* **Vitrine de Serviços**: Organizada em um sistema de grid (3x2) com cards detalhados.
* **Mural de Depoimentos**: Layout estilo "masonry" (bagunçado/orgânico) com prints reais de clientes do Google, utilizando inclinações via CSS (`transform: rotate`).
* **Seção de Contato**: Divisão intuitiva entre informações da barbearia e formulário de contato funcional.
* **Navegação Fluida**: Links de âncora para facilitar a navegação interna.
